# Mini web (PWA) - Travel Hub Madrid & Lisboa

Mini web **mobile-first** y **compartible por link**.

## Cómo editar
Edita `data.json` y (opcional) copia PDFs/imágenes a `assets/`.

## Publicar (para que lo abran varias personas)
Hospédalo en un sitio estático (GitHub Pages o Netlify).

## Uso en celular
iPhone (Safari): Compartir → Agregar a pantalla de inicio.
Android (Chrome): Menú ⋮ → Agregar a pantalla principal.
